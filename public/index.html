Pengo

(PEN-go) noun

  1. Classic Sega arcade game
  2. Infamous German KGB hacker Hans Heinrich Hübner
  3. Your coding buddy on Slack!

Supported Commands

/pengo bash <command>

get information about the command and its syntax

/pengo

get a random chunk of wisdom from pengo

/pengo [ID]

get a specific chunk

/pengo help

learn how to use pengo

/pengo rant

get the ultimate push
